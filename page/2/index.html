URI,URL与URN的区别

2018-03-23

URI：统一资源标识符 Uniform Resource Identifier
URL：统一资源定位符 Uniform Resource Locator
URN：统一资源名称 Uniform Resource Name

URI通过标识符的方式确定一个资源。
URL通过定位的方式确定一个资源。
URN通过名称的方式确定一个资源。
URL和URN是URL的子集。URI可以是URL，URN或者两者都是。
