gun global使用笔记

Feb 3, 2016 | 阅读

global是一个由一家日本公司开发的源代码tag系统。和ctags是一类的工具。不过比ctags要强大很多，具体的对比可以参考这个表格：(表格地址)。

安装

mac：

brew install global

centos源码安装：

一开始我在运行sh reconf.sh期间我遇到了错误：

`COPYING.LIB' not found in `/usr/share/libtool/libltdl'

这是因为没有安装libtool-ltdl-devel。参考帖子。

生成tag文件

$ cd source_code_dir
$ gtags

global会遍历改目录下的所有子目录，处理所有代码文件，然后在当前目录下生成三个文件：

$ ls G*
GPATH   GRTAGS  GTAGS

这三个文件：

  • GTAGS 定义数据库
  • GRTAGS 引用数据库
  • GPATH 路径名数据库
