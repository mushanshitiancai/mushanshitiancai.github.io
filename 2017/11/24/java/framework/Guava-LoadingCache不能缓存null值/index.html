Guava LoadingCache不能缓存null值

Nov 24, 2017 | Java | 阅读

测试的时候发现项目中的LoadingCache没有刷新，但是明明调用了refresh方法了。后来发现LoadingCache是不支持缓存null值的，如果load回调方法返回null，则在get的时候会抛出异常。
